# ansible-linux-setup
Initial Linux distro setup tasks.
For usage on locahost.
VPN setup: https://protonvpn.com/support/linux-vpn-setup/
Graphics Driver Package: nvidia-driver

## Usage
```ansible-playbook -vK pc.yml```
